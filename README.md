## Preparando Ambiente de desarrollo

Primero instala el proyecto:
```bash
pnpm i
```

Para correr el proyecto:

```bash
pnpm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver los resultados
